# Expense Tracker CLI

A command-line application to track your expenses, set budgets, and view summaries -- built in Python

---

## Features
- Add, update, delete, and list expenses
- Categorize expenses
- View monthly and category summaries 
- Export expenses or budgets to CSV (WIP)
- Automatically handles JSON persistence 

--- 

## Requirements
- Python 3.10+ 
- Install dependencies:
```bash
  pip install -r requirements.txt
```

## Usage
Run the CLI directly from your terminal: 
```bash 
python expense-tracker.py [command] [options]
```
